## How to use this tracker

1. Run each query in each engine with web search enabled.
2. For each cell:
   - ✓ if FlexForce is explicitly cited / linked in the answer
   - △ if FlexForce is mentioned without a direct link / footnote
   - ✗ if FlexForce is not mentioned at all
   - C:[name] if a competitor was cited (Workrise, BlueCrew, VeroSkills, Workiz, ServiceTitan, Indeed, Mya, Paradox, Jobber, etc.)
3. In the **Notes** column, capture:
   - Which source page the engine cited (Crunchbase, G2, Reddit, your own URL)
   - Whether the cited source is one FlexForce has presence on
   - Anything unusual or worth investigating
4. After ~6 weeks of weekly logging, you'll have enough data to see which off-site investments are paying off and which queries to optimize next.

## Cadence

- **Every Monday morning, 30 minutes.** Block on calendar; consistency matters more than depth.
- **Monthly retro (first Monday of month, 1 hour).** Compare against the prior 4 weeks; pick 1–2 high-leverage actions for the month.
- **Quarterly check-in.** Refresh `ai-visibility-baseline.md` with the empirical patterns observed.
